# KrishiBodhi AI

An AI-powered offline-first chatbot providing instant regional crop advisory to high-altitude field supervisors in Uttarakhand.

## Tech Stack
- **Frontend:** React
- **Styling:** Tailwind CSS
- **Backend:** FastAPI (Python)
- **Database:** MongoDB
- **Deployment:** Vercel + Render
